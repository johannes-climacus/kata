# Description

Write a program that prints the integers from 1 to 100.

# Requirements

*But:*

- For multiples of three, print **Fizz**
- For multiples of three, print **Buzz**
- For multiples of three and five, print **FizzBuzz**

# Examples

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
FizzBuzz
```
